近年、Vision-Language-Action Model（VLA）やVision-Language Model（VLM）を用いたロボット操作は、物体を掴む、置く、移動するなどの短期操作で大きく進展している。
一方で、机上片付けや家庭内支援のような長期ロボットタスクでは、現状の手法は次の動作を生成することには有効であっても、現在の工程において何が不確実であり、その不確実性にどう対処すべきかを適切に判断することが難しい。
本研究では、VLMを次行動生成器としてではなく、工程文脈依存的不確実性を診断するモジュールとして用いる。
特に、同じ「見えない」「状態が分からない」という状況でも、現在工程や作業履歴によって、実行、再観測、人間確認、安全停止、後回しのいずれが適切かは変化する。
現在画像、作業履歴、現在工程、次工程候補、工程条件リストを入力とし、evidence sufficiency、uncertainty source、task relevance、resolution mode、execution guard を構造化出力させることで、不確実性を act / reobserve / ask\_user / stop / defer の分岐へ接続する方法を検討する。
評価では、同一視覚状態に対して工程文脈のみを変化させた対照ペアを設計し、VLMが文脈に応じて分岐判断を切り替えられるかを検証する。
また、Zero-shot VLM や簡略化条件との比較により、構造化診断の有効性を評価する。
さらに、実機デモを通じて、診断結果が安全な分岐制御に接続可能かを確認する。
本研究により、ロボットが「分からない」状態を一律に扱うのではなく、工程文脈に応じて、実行する、見直す、聞く、止まる、後回しにする判断を切り替えるための基礎的知見を得ることを目指す。
